# CROSSMIX

イベントで使う、**スマホ専用のプロフィール（名刺）交換アプリ**です。
参加者がプロフィールを登録すると、全員の画面にカードがリアルタイムで並びます。タグごとに分かれて表示され、カードをタップすると詳細（SNS・HPリンクなど）が見られます。

## 主な機能

- **プロフィール登録**：写真・名前・所属・一言・タグ・SNS / HPリンク
- **一覧表示**：登録されたカードを新着順・タグ別セクションで表示（リアルタイム反映）
- **詳細モーダル**：カードをタップすると拡大表示。入力されたリンクだけボタンで表示
- **「すべて表示」**：同じタグに6人以上いると、最初の5枚＋「すべて表示」カードを出す
- **写真**：長辺320pxに縮小し、文字データ(JPEG)として Firestore に直接保存（Storage 不使用＝無料プランで完結）

## 使っている技術

- HTML / CSS / JavaScript（jQuery）
- Firebase Firestore（モジュール版 v12.14.0）でデータを保存・リアルタイム表示

## 動かし方（ローカル）

1. VS Code の **Live Server** などで `index.html` を開く
2. ブラウザ（スマホ表示推奨）でプロフィールを登録すると、カードが並ぶ

> ⚠️ **APIキーについて**
> このリポジトリでは、`index.html` の `firebaseConfig` から **APIキー（apiKey）を外しています**。
> そのため、このままでは Firebase につながりません。
> 手元で動かすときは、`firebaseConfig` の中に `apiKey: "（自分のキー）"` の行を追加してください。
> （Firebase の Web キーは公開前提の値ですが、提出ルール上ここでは外しています）

## メモ

- データは Firestore の `profiles` コレクションに1件ずつ保存されます。
- リセットしたいときは、Firebase コンソールの Firestore で `profiles` の中身を削除します。
- Firestore のセキュリティルールは現在「テストモード（誰でも読み書き可）」です。**一般公開する前にルールを締める**必要があります。
